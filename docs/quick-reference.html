ML Quick Reference

MetricFormula / Use
PrecisionTP / (TP + FP) — "of positive predictions, how many were correct?"
RecallTP / (TP + FN) — "of actual positives, how many did we find?"
F1 Score2 * (Precision * Recall) / (Precision + Recall)
AUC-ROCArea under ROC curve; 0.5 = random, 1.0 = perfect
MSEMean Squared Error — for regression
R²1 - (SS_res / SS_tot) — variance explained by model

LLM & Agent Patterns

PatternDescription
ReActThought → Action → Observation loop for reasoning agents
RAGRetrieve relevant docs, inject into prompt as context for grounded answers
Function callingModel outputs structured JSON for tool invocation, code executes the tool
Circuit breakerStop calling a failing service after N failures; retry after timeout
Fallback chainTry primary model → fallback model → cached response
Model routingSimple queries → small model; complex → large model
Human-in-the-loopRequire human approval for high-stakes agent decisions

API Key Management

RuleDetails
Never hardcode keysUse .env files + python-dotenv or os.environ
.gitignoreAlways include .env, *.key, credentials.json
ProductionUse secrets managers (AWS, GCP, Azure Key Vault)
RotationRotate keys regularly; design for zero-downtime rotation
ScopingUse project-specific keys with minimum permissions
